Web bajo mi punto de vista y lo que sé

HTML

Como primer herramienta a la hora de desarrollar una web tenemos un lenguaje de estructuración llamado html el cual nos ayuda a dar una esetructura básica de una pagina web, html se usa por medio de etiquetas, las cuales forman elementos o también conocidos como cajas, nuestro trabajo como programadores es dar un orden a esas cajas para luego pasar a la siguiente parte.

CSS

CSS es la siguiente herramienta que usaremos, esta nos ayuda a dar estilo y retocar la forma de nuestro html, con esta herramienta podemos dar colores, sombras, formas e incluso algunas animaciones que le podrán dar mejor presencia a nuestro sitio web, cabe recalcar que las dos herramientas mencionadas no son consideradas lenguajes de programación ya que no requieren de cierta lógica que los lenguajes de programación usan.

JAVASCRIPT

JAVASCRIPT es un lenguaje de programación interpretado, es decir que no necesita de un compilador sino que más bien se puede visualizar en tiempo real por medio de un navegador, el campo más fuerte donde se usa JAVASCRIPT es web pero también hay formas de usarlo sin navegadores y por medio de tecnologías como pueden ser NODE.js o VUE.js, este lenguaje en web se usa principalmente para fronted, sin embargo eso no significa que no se use para backend, pero para backend tenemos otra opción que viene a continuación.

PHP

PHP es un lenguaje de programación el cual en un inicio se creo para dar vida al backend de la web, por lo cual en la mayoría de casos este solo se trabaja en web, PHP tiene abarca un gran porcentaje de la web, incluso el 40% de los sitios web están hechos en WORDPRESS y este esta hecho en PHP por lo cual es un lenguaje del cual encontrarás un montón de información, como se dijo en el inicio este lenguaje está enfocado para principalmente para backend.

MYSQL

MYSQL es un lenguaje de base de datos relacional, en el cual nosotros podremos dar una estructura de base de datos a nuestra web, hay muchas variantes de sql, pero la mayoría de estas variantes trabajan con la misma sintaxis de SQL. Algunos ejemplos son:

  • MariaDB
  • MongoDB
  • SQLite
  • SQL server
  • Kafka
